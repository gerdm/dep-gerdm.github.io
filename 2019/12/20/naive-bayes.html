Naive Bayes

Naive Bayes es un modelo de machine learning para clasificar textos entre una serie de categorías. Ejemplos de esto podrían ser libros de un autor, tweets de alguna persona, diálogos de personajes dentro de un texto, etc.

La bondad de este modelo es que a pesar de ser muy simplista, en la práctica puede arrojar resultados de gran calidad.

Para poder hacer uso del modelo, lo primero es contar con una base de datos con documentos, cada uno asignado a una categoría (o clase). Visto de otra forma, sea dnd_ndn​ el nnn-ésimo documento de nuestra base y, cnc_ncn​ la clase perteneciente al documento dnd_ndn​, entonces nuestra base de datos estaría conformada por D={(dn,cn)}n=1N\mathcal D = \{(d_n, c_n)\}_{n=1}^ND={(dn​,cn​)}n=1N​. La representación matemática de un documento dnd_ndn​ está dado por un vector con las palabras únicas del documento, esto es d=(w1,…,w∣V∣n)d=(w_1, \ldots, w_{|V|_n})d=(w1​,…,w∣V∣n​​); donde ∣V∣n|V|_n∣V∣n​ denota la cardinalidad de dnd_ndn​.

Dado un documento ddd fuera de nuestra base de datos, queremos clasificarlo entre alguna de las KKK clases. Una manera razonable de lograr eso sería eligiendo aquella clase con mayor probabilidad de la clase dado el documento. Esto es,

k^=arg⁡max⁡kp(Ck∣d)=arg⁡max⁡kp(Ck)p(d∣Ck)p(d)=arg⁡max⁡kp(Ck)p(d∣Ck). \begin{aligned} \hat k &= \arg\max_k p(C_k|d)\\ &= \arg\max_k \frac{p(C_k)p(d|C_k)}{p(d)}\\ &= \arg\max_k p(C_k)p(d|C_k). \end{aligned} k^​=argkmax​p(Ck​∣d)=argkmax​p(d)p(Ck​)p(d∣Ck​)​=argkmax​p(Ck​)p(d∣Ck​).​

Es decir, la probabilidad de clasificar un nuevo documento ddd está dado por el valor máximo que resulta en calcular la probabilidad a priori de una clase kkk multiplicado por la verosimilitud del documento dada la clase kkk.

El siguiente paso sería definir quién es p(Ck)p(C_k)p(Ck​) y quién p(d∣Ck)p(d|C_k)p(d∣Ck​). Para p(Ck)p(C_k)p(Ck​) un valor a priori razonable sería considerar la proporción de textos que pertenecen a CkC_kCk​. Sea NkN_kNk​ el número de documentos pertenecientes a la clase kkk y ND=∑kNkN_D = \sum_k N_kND​=∑k​Nk​. Entonces, definimos la probabilidad a priori de un documento kkk como

p(Ck)=NkND. p(C_k) = \frac{N_k}{N_D}. p(Ck​)=ND​Nk​​. Por otro lado, calcular p(d∣Cl)=p((w1,…,w∣V∣n)∣Cl)p(d|C_l) = p\left((w_1, \ldots, w_{|V|_n})|C_l\right)p(d∣Cl​)=p((w1​,…,w∣V∣n​​)∣Cl​) implica obtener la probabilidad del conjunto de palabras. Es en este momento que el modelo de Naïve Bayes toma forma. A fin de obtener una verosimilitud para ddd, haremos dos grandes suposiciones sobre las propiedades de p(d∣Ck)p(d|C_k)p(d∣Ck​)

  1. La suposición de la bolsa de palabras: En primera instancia, asumiremos que el orden de las palabras, condicionado a una clase, no son de importancia, es decir, para dos palabras wiw_iwi​, wjw_jwj​, p(wi,wj∣Ck)=p(wj,wi∣Ck)p(w_i, w_j|C_k) = p(w_j, w_i|C_k)p(wi​,wj​∣Ck​)=p(wj​,wi​∣Ck​)

  2. La suposición de Naive Bayes: Condicionado a una clase kkk, cada una de las palabras son independientes. Es decir, dada dos palabras wiw_iwi​, wjw_jwj​, p(wi,wj∣Ck)=p(wj∣Ck)p(wi∣Ck)p(w_i, w_j|C_k) = p(w_j|C_k) p(w_i|C_k)p(wi​,wj​∣Ck​)=p(wj​∣Ck​)p(wi​∣Ck​)

Con estas dos suposiciones podemos decir que dado un nuevo documento ddd , lo clasificamos como la clase

k^=arg⁡max⁡kp(Ck)∏v∈Vdp(wv∣Ck)=arg⁡max⁡klog⁡p(Ck)+∑v∈Vdlog⁡p(wv∣Ck) \begin{aligned} \hat k &= \arg\max_k p(C_k) \prod_{v \in V_d} p(w_v|C_k)\\ &= \arg\max_k \log p(C_k) + \sum_{v\in V_d } \log p(w_v|C_k) \end{aligned} k^​=argkmax​p(Ck​)v∈Vd​∏​p(wv​∣Ck​)=argkmax​logp(Ck​)+v∈Vd​∑​logp(wv​∣Ck​)​

donde VdV_dVd​ es el vocabulario del documento ddd.

Estimando p(wv∣Ck)p(w_v|C_k)p(wv​∣Ck​)

La probabilidad p(wv∣Ck)p(w_v|C_k)p(wv​∣Ck​) la conseguiremos a partir de máxima verosimilitud. Considera

p(wv∣Ck)=p(wv,Ck)p(Ck)=p(wvk)p(Ck)=p(wvk)∑v∈Vkp(wvk) \begin{aligned} p(w_v|C_k) &= \frac{p(w_v, C_k)}{p(C_k)}\\ &= \frac{p(w_v^k)}{p(C_k)}\\ &= \frac{p(w_v^k)}{\sum_{v\in V_k}p(w_v^k)} \end{aligned} p(wv​∣Ck​)​=p(Ck​)p(wv​,Ck​)​=p(Ck​)p(wvk​)​=∑v∈Vk​​p(wvk​)p(wvk​)​​

donde denotamos wvk=(wv,Ck)w_v^k = (w_v, C_k)wvk​=(wv​,Ck​).

Asumamos wvk∼Bern(μvk)w_v^k \sim \text{Bern}(\mu_v^k)wvk​∼Bern(μvk​). Luego, la máxima verosimilitud de μvk\mu_v^kμvk​ estaría dada por

μ^vk=arg⁡max⁡μvkp(D∣μvk)=arg⁡max⁡μvk∏np(dn∣μvk)=arg⁡max⁡μvk∏wm∈Wp(wm∣μvk) \begin{aligned} \hat \mu_v^k &= \arg\max_{\mu_v^k } p(D|\mu_v^k) \\ &= \arg\max_{\mu_v^k } \prod_n p(d_n|\mu_v^k) \\ &= \arg\max_{\mu_v^k } \prod_{w_m\in W} p(w_m|\mu_v^k) \\ \end{aligned} μ^​vk​​=argμvk​max​p(D∣μvk​)=argμvk​max​n∏​p(dn​∣μvk​)=argμvk​max​wm​∈W∏​p(wm​∣μvk​)​

Notemos que la primera y segunda igualdad corren sobre cada documento y la última igualdad corre sobre cada palabra, dentro de cada documento.

Antes de proseguir, denotemos 1(n)v,k=1(wn=wv)∨(wn∈Vk)\bold{1}(n)_{v,k} = \bold{1}_{(w_n = w_v) \vee (w_n \in V_k)}1(n)v,k​=1(wn​=wv​)∨(wn​∈Vk​)​, la indicadora de una palabra wmw_mwm​ dentro de la base de datos que sea igual a wvw_vwv​ y que pertenezca al vocabulario de elementos de la clase kkk. Entonces, μ^vk=arg⁡max⁡μvk∑mlog⁡p(wm∣μvk)=arg⁡max⁡μvk∑m1(n)v,klog⁡μvk+(1−1(n)v,k)log⁡(1−μvk)=arg⁡max⁡μvkC(wvk)log⁡μvk+(1−C(wvk))log⁡(1−μvk)=C(wvk)N \begin{aligned} \hat \mu_v^k &= \arg\max_{\mu_v^k } \sum_m \log p(w_m|\mu_v^k) \\ &= \arg\max_{\mu_v^k } \sum_m \bold{1}(n)_{v,k} \log \mu_v^k + (1 - \bold{1}(n)_{v,k})\log(1 - \mu_v^k)\\ &= \arg\max_{\mu_v^k } C(w_v^k) \log \mu_v^k + (1 - C(w_v^k))\log(1 - \mu_v^k)\\ &=\frac{C(w_v^k)}{N} \end{aligned} μ^​vk​​=argμvk​max​m∑​logp(wm​∣μvk​)=argμvk​max​m∑​1(n)v,k​logμvk​+(1−1(n)v,k​)log(1−μvk​)=argμvk​max​C(wvk​)logμvk​+(1−C(wvk​))log(1−μvk​)=NC(wvk​)​​

Dónde C(wvk)C(w_v^k)C(wvk​) es un contador del número de palabras de wvw_vwv​ dentro de documentos de la clase kkk.

Con este último resultado podemos ver que p(wv∣Ck)=C(wvk)/N∑v^∈VC(wv^k)/N=C(wvk)∑v^∈VkC(wv^k) \begin{aligned} p(w_v|C_k) &= \frac{C(w_v^k)/N}{\sum_{\hat v\in V}C(w_{\hat v}^k)/N}\\ &= \frac{C(w_v^k)}{\sum_{\hat v\in V_k}C(w_{\hat v}^k)} \end{aligned} p(wv​∣Ck​)​=∑v^∈V​C(wv^k​)/NC(wvk​)/N​=∑v^∈Vk​​C(wv^k​)C(wvk​)​​

Es decir, la probabilidad de una palabra wvw_vwv​ dado un documento de la clase kkk está dado por cuántas veces aparece la palabra wvw_vwv​ dentro de la clase kkk, divido por el total de palabras que se dijeron dentro de la clase kkk.

Parecería entonces que ya tenemos un modelo sobre el cuál podemos estimar una clase dado un documento ddd, pero, ¿qué pasa si el documento ddd tiene información relevante sobre una clase k^\hat kk^ y exista una palabra w^j\hat w_jw^j​ que no existe dentro de VkV_kVk​? En este caso, p(w^j∣Vk)p(\hat w_j|V_k)p(w^j​∣Vk​) sería cero y, subsecuentemente, p(Ck∣d)=0p(C_k|d) =0p(Ck​∣d)=0. Para evitar esta clase de problemas, podemos suavizar las probabilidades de por el método de la suavización de Laplace (Laplace Smoothing)

La suavización de Laplace

Consideremos nuevamente wvk∼Bern(μvk)w_v^k\sim \text{Bern}(\mu_v^k)wvk​∼Bern(μvk​). Asumiremos a priori μvk∼U(0,1)\mu_v^k\sim U(0,1)μvk​∼U(0,1). Entonces, queremos calcular la probabilidad a posteriori de μvk\mu_v^kμvk​. Esto es

p(μvk∣D)∝p(D∣μvk)p(μvk)=(μvk)C(wvk)(1−μvk)N−C(wvk) \begin{aligned} p(\mu_v^k|\mathcal{D}) &\propto p(\mathcal{D}|\mu_v^k) p(\mu_v^k)\\ &=(\mu_v^k)^{C(w_v^k)} (1 - \mu_v^k)^{N - C(w_v^k)} \end{aligned} p(μvk​∣D)​∝p(D∣μvk​)p(μvk​)=(μvk​)C(wvk​)(1−μvk​)N−C(wvk​)​

Con esto último podemos ver que μvk∣D∼Beta(C(wvk)+1,N−C(wvk)+1). \mu_v^k|\mathcal{D}\sim\text{Beta}(C(w_v^k) + 1, N - C(w_v^k) + 1). μvk​∣D∼Beta(C(wvk​)+1,N−C(wvk​)+1). Cuya esperanza está dada por E[μvk∣D]=C(wvk)+1N+2. \mathbb{E}[\mu_v^k|\mathcal{D}] =\frac{C(w_v^k) + 1}{N + 2}. E[μvk​∣D]=N+2C(wvk​)+1​.

Este último resultado sería una manera alternativa a elegir μvk\mu_v^kμvk​ dado por máxima verosimilitud. De esta forma

p(wn∣Ck)=C(wvk)+1∑v^∈VkC(wv^k)+∣V∣ p(w_n|C_k) = \frac{C(w_v^k) + 1}{\sum_{\hat v\in V_k}C(w_{\hat v}^k) + |V|} p(wn​∣Ck​)=∑v^∈Vk​​C(wv^k​)+∣V∣C(wvk​)+1​

garantiza una probabilidad (aunque muy pequeña) para cada palabra que veamos el algún nuevo documento.

Finalmente, nuestro método para determinar la clase kkk de un documento estaría dado por: k^=arg⁡max⁡klog⁡NkND+∑v∈Vdlog⁡C(wvk)+1∑v^∈VkC(wv^k)+∣V∣ \hat k= \arg\max_k \log \frac{N_k}{N_D} + \sum_{v\in V_d } \log \frac{C(w_v^k) + 1}{\sum_{\hat v\in V_k}C(w_{\hat v}^k) + |V|} k^=argkmax​logND​Nk​​+v∈Vd​∑​log∑v^∈Vk​​C(wv^k​)+∣V∣C(wvk​)+1​
